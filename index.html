About Me

Hello and welcome! My name is Aaron Park and I'm a Web Developer residing in Irvine, CA.

I graduated from the Unversity of California: Irvine with earining a Bachelor's degree in Criminology. In time, I obtained a desire for programming and attended LearningFuze's Accelerated Web Development Program.

The appeal of programming for me is having the ability to create any project I want. The reasons for doing so can range from skill development to just creating a useful website for a niche hobby. Creating a new site or improving on an existing site is no longer some wistful dream but rather a practical goal.

My greatest programming achievement is just being able to do what I can do now. Before I attended LearningFuze, I had zero skills in programming let alone web development. Now, I am able to create a website from scratch, along with its required APIs and database, and host it on a server. It is difficult for me to not be in awe of my own skill growth.

In my off time, I'm either cooking, practicing the piano, or playing obscure Japanese arcade rhythm games.
